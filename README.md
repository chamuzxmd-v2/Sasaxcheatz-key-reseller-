Advanced **Key Management System** with Dual-Layer Authentication (Password + GitHub OAuth) and a Cyberpunk Neon Matrix aesthetic. Built for resellers who need to manage license keys live from a web-based terminal.

## 🚀 Features
* **Dual-Stage Security:**
    1.  Master Admin Password Authentication.
    2.  Real-time GitHub OAuth Verification.
* **Live Key Management:** Add new keys and reset existing ones instantly from the dashboard.
* **Matrix Aesthetic:** Smooth Green Neon Matrix background animation.
* **SQLite Database:** No complex SQL server setup needed; the DB is a single file.
* **Responsive UI:** Works on both PC and Mobile terminals.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Animations), JavaScript.
* **Backend:** PHP (PDO).
* **Database:** SQLite 3.
